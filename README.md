# LINDA - صفحة هبوط منتج العناية بالبشرة

صفحة هبوط فاخرة واحترافية لمنتج LINDA - أول عجينة تزيل الشعر بدون ألم نهائي.

## المميزات

- تصميم فاخر وأنيق بأسلوب Glossier و Dior Beauty
- ألوان ناعمة (وردي، أبيض، بيج فاتح)
- متجاوب على جميع الأجهزة (Responsive Design)
- أقسام متعددة: البطل، المميزات، طريقة الاستخدام، التقييمات، السعر
- رمز QR يربط بحسابات التواصل الاجتماعي
- أزرار CTA واضحة للشراء والتواصل

## الملفات

- `index.html` - ملف صفحة الهبوط الرئيسي
- `assets/images/` - مجلد الصور (صورة المنتج ورمز QR)
- `social_links.html` - صفحة روابط التواصل الاجتماعي
- `content.md` - المحتوى التسويقي

## كيفية الاستخدام

1. فك ضغط الملفات
2. افتح ملف `index.html` في متصفح الويب
3. أو رفع الملفات على خدمة استضافة (GitHub Pages, Netlify, إلخ)
